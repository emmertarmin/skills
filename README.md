# Collection of AI Coding Agent Skills I created for my personal use.

## [Redmine API](./redmine-api/SKILL.md) - A skill to interact with Redmine's REST API for project management tasks.

Prerequisites:
- Redmine account and API access token.
- Append the following to your ~/.bashrc:

```bash
export REDMINE_API_KEY="your_redmine_api_key"
export REDMINE_API_URL="https://your_redmine_instance_url"
```

You might need to source your .bashrc or restart your terminal for the changes to take effect: `source ~/.bashrc`

## [Sentry CLI](./sentry-cli/SKILL.md) - A skill to interact with Sentry's CLI for error tracking and monitoring.

Prerequisites:
- Sentry account and API token.
- Sentry CLI installed and configured on your system.
